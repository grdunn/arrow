Typography

The base font-family is Roboto via Google Fonts. It's set in REMs, where the html element is set to 62.5%. All measurements are base 10, so 1rem is equal to 10px

Grid System

A standard 12 column flex grid. The gutter is controlled by a $grid-gutter variable that is set in percentages. It breaks down to full width at the $grid-breakdown variable.
